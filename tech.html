  • 팀 소개
  • 프로젝트 소개

ChatGPT

  • 1. 사용자 입력 → 자연어 토큰화
  • 2. Transformer 모델을 통해 문맥 이해
  • 3. 확률 기반으로 문장 예측 및 생성

DALL·E 2

  • 1. 텍스트 프롬프트 → CLIP으로 임베딩
  • 2. 디퓨전 모델을 통해 이미지 점진 생성
  • 3. 고해상도 이미지로 리파인

Midjourney

  • 1. 감성 기반 텍스트 임베딩
  • 2. 스타일 특화 디퓨전 모델 사용
  • 3. 예술적 이미지 생성

작동 원리 시각화

DALL·E 2: 텍스트 → 임베딩 → 디퓨전 → 이미지

Midjourney: 프롬프트 → 디퓨전 → 스타일 이미지

ChatGPT: 입력 → 문맥 이해 → 문장 생성

기술 비교 표

도구 사용 기술 특징 요약
DALL·E 2 GPT, CLIP, 디퓨전 텍스트를 현실 이미지로 변환
Midjourney 디퓨전 기반 예술적 스타일의 감성 이미지
ChatGPT GPT (Transformer) 자연어 이해 및 대화 생성
